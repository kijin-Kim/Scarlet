\doxysubsection{Detailed Description}
입력 액션과 게임플레이 태그사이의 관계를 정의하고 표현합니다. 

게임플레이 태그와 로직을 연관시켜 유연성을 증가시킵니다. 입력 액션과 로직 사이에는 게임플레이 태그가 존재하기 때문에, 로직에 대한 변경을 최소화 할 수 있습니다.
